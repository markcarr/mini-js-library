miniLib - a (very) basic mini javascript library

miniLib has been tested in the following browsers:

  • IE7-10
  • Firefox
  • Chrome

DOM selector

Examples:

ID selector

$(#id);

Class selector

$(.classname);

Class and tag selector

$(.classname, tagname);

Tag selector

$(tagname);

addEvent

Example: Click event 
addEvent(element, event, function(event){
	// Code goes here
});

Events

  • click
  • blur
  • focus
  • mouseover
  • mouseout
  • touchstart
  • touchend

Prevent default event

stop(event);

Add classname to element

Example: Add class 
addClass(element, classname);

Remove classname from element

Example: Remove class 
removeClass(element, classname);

Toggle classname on element

Example: Toggle class 
toggleClass(element, classname);

Check to see if element has classname - boolean

Example: Has class 
hasClass(element, classname);

For each looping

[].forEach.call(array, function(el){
	// Code goes here
});
